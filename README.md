## 📂 Repository Structure

This repository acts as an automated portfolio, integrated directly with LeetCode via **LeetSync**. 

Whenever a solution passes all test cases and is marked as "Accepted," a new directory is automatically generated containing:
1. **The Source Code:** The highly optimized `Java` solution file.
2. **Problem Documentation:** A dedicated `README.md` containing the full problem statement, constraints, and examples.
3. **Performance Metrics:** A commit record detailing the solution's exact Time and Space Complexity (e.g., *Beats 100% of users in runtime*).
